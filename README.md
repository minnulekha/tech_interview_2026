## 🚀 Project Enhancements & Features Completed

As part of the Tech Team Challenge, this repository has been successfully enhanced with the following features:

* **Interactive Contact Section:** Added a sleek, slide-out drawer containing a fully structured contact form and social reach-out links.
* **Expanded Event Menu:** Added 4 beautifully aligned event cards (Python Masterclass, React Foundations, Git Flow, and Networking) using CSS Grid/Flexbox, complete with thumbnail images.
* **Custom Registration Page:** Created a dedicated `register.html` page indicating that "Registrations are currently closed."
* **"DEBUG RUSH" Mini-Game:** Built an interactive, terminal-style JavaScript mini-game directly into the registration page. Users can select between JavaScript and Python runtime environments to fix buggy code snippets against a timer while they wait for registrations to open.

## Workflow: How to Clone and Contribute

Follow these steps to set up the project on your local machine and start contributing:

### Fork the Repository
* Navigate to the official repository.
* Click the Fork button to create a copy in your GitHub account.

### Clone to Your Local System
Open your terminal and run the following command to clone your forked repository:

```bash
git clone [https://github.com/minnulekha/tech_interview_2026.git](https://github.com/minnulekha/tech_interview_2026.git)
```

Navigate to the project directory:

```bash
cd tech_interview_2026
```

### Make Changes
Open the code in your preferred text editor, create a new branch, and make your necessary enhancements or bug fixes.

### Add, Commit, and Push
Once you are satisfied with your changes, stage and commit them with a descriptive message:

```bash
git add .
git commit -m "Add descriptive commit message here"
```

Push the changes back to your GitHub repository:

```bash
git push origin main
```

### Create a Pull Request
* Go to your forked repository on GitHub.
* Click on **Compare & pull request**.
* Add a clear title and description of your changes, then submit the Pull Request for review.
